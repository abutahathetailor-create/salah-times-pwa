# Salah Times PWA

A Progressive Web App that displays real-time prayer times for Jubail, Saudi Arabia.

## Features
- Real-time prayer times for Jubail
- Countdown to next prayer
- PWA - Installable on mobile devices
- Works offline with fallback data
- Beautiful responsive design

## Installation

### For Mobile Users:
1. Open this website in Chrome (Android) or Safari (iPhone)
2. Tap the share button
3. Select "Add to Home Screen"
4. The app will install like a native app

### For Web:
Simply visit the website - it works in all modern browsers.

## Technical Details
- Uses Aladhan API for prayer times
- PWA with service worker for offline functionality
- Responsive design for all screen sizes
- Automatic updates

## Files
- `index.html` - Main application
- `manifest.json` - PWA manifest
- `sw.js` - Service worker
- `icons/` - App icons

## Deployment
Upload all files to any web hosting service or GitHub Pages.
